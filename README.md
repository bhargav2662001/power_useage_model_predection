# Electricity Consumption Prediction App

A machine learning application that predicts next-hour electricity usage based on previous consumption patterns using Linear Regression.

## 📋 Overview

This project uses historical power consumption data to predict future electricity usage. The application provides:
- Real-time power consumption predictions
- Usage categorization (Low/Medium/High)
- Energy-saving recommendations

## 🚀 Features

- **Predictive Modeling**: Linear Regression model trained on historical power consumption data
- **Interactive UI**: Built with Streamlit for easy user interaction
- **Smart Categorization**: Automatically classifies usage into three categories
- **Actionable Insights**: Provides energy-saving suggestions based on predicted usage

## 🛠️ Technologies Used

- **Python** - Core programming language
- **Scikit-learn** - Machine learning model (Linear Regression)
- **Streamlit** - Web application framework
- **Joblib** - Model persistence
- **NumPy** - Numerical computations
- **Pandas** - Data manipulation
